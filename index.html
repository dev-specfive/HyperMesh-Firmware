üöÄ Flashing the Firmware

Prerequisites

Before flashing, ensure you have:

  • A compatible Specfive device (MiniTrekker or Trekker Bravo)
  • USB cable for connecting the device to your computer
  • Appropriate USB drivers installed (e.g. CP210x & CH340)

Download the Firmware

Select the appropriate firmware file for your device:

  • MiniTrekker ‚Üí firmware_heltec_wireless_v3_1.bin
  • Trekker Bravo ‚Üí firmware_heltec_tracker_v1_2.bin

Required Tools

You will need the ESP32 Flash Download Tool to flash the firmware onto your device.

üîó Download ESP32 Flash Download Tool

Enter Download Mode

Specfive Trekker Bravo

You must put the device in download mode before the flash tool can connect. Follow these steps:

  1. Connect the device to your computer via USB cable
  2. Press and hold the USER button
  3. While holding USER, press the RST (reset) button once
  4. Release the USER button
  5. The device is now in download mode and ready for firmware flashing

Perform this sequence right before clicking START in the flash tool. If flashing fails to begin, repeat the steps and try again.

Specfive Mini Trekker

Typically enters download mode automatically when the flash tool connects; no button press required.

Step-by-Step Flashing Instructions

Reference: ESP32-S3 Flash Download Tool V3.9.9_R2 - Successful flash completion

  1. Connect Your Device
    Connect your Specfive MiniTrekker or Specfive Trekker Bravo to your computer via a USB cable.
  2. Open Flash Tool
    Launch the ESP32-S3 Flash Download Tool on your computer. Select ESP32-S3 as the hardware type.
  3. Select COM Port
    Choose the COM port (Windows) or Serial port (macOS/Linux) where your device is connected. The tool will automatically detect the port.

    To find your connected Heltec device's COM port on Windows, open Device Manager ‚Üí expand Ports (COM & LPT) ‚Üí look for USB Serial Device (or CH340/CP210x depending on your board). The number in parentheses (e.g., COM39) is the port to select.

    Reference: Finding your Heltec device's COM port in Windows Device Manager (Ports ‚Üí USB Serial Device)

  4. Configure Flash Settings
    Set the following parameters:
    • Flash Address: 0x000
    • Hardware: ESP32-S3
    • SPI Speed: 40MHz (recommended)
    • SPI Mode: DIO (default)
    • Baud Rate: 921600 (or lower if experiencing issues)
  5. Load Firmware
    Click Browse or Add button and navigate to select the downloaded firmware file:
    • firmware_heltec_wireless_v3_1.bin for MiniTrekker
    • firmware_heltec_tracker_v1_2.bin for Trekker Bravo
    Ensure the checkbox next to the firmware file is checked, and verify the flash address shows 0x000.
  6. Start Flashing
    Click START to begin uploading the firmware to your device. Wait for the flashing process to complete (progress bar will show status). You should see a "FINISH" message (ÂÆåÊàê) when successful, as shown in the reference image above.
  7. Reboot Device
    After the flashing process completes:
    1. Disconnect the USB cable
    2. Power cycle the device (turn off and on)
    3. The device should now be running the new firmware

üîß Troubleshooting

Device Not Detected

  • Ensure the correct USB drivers for your device are installed
  • Try a different USB cable (data-capable cable required)
  • Try a different USB port on your computer
  • Check Device Manager (Windows) or System Information (macOS) for COM port detection

Flashing Fails to Start

  • Trekker Bravo: Put device in download mode ‚Äî hold USER, press RST once, then release USER. Repeat before clicking START if needed.
  • MiniTrekker: Typically enters download mode automatically when the flash tool connects; no button press required.
  • Ensure the device is in download mode before starting
  • Check that the correct COM port is selected
  • Verify the firmware file is not corrupted (re-download if necessary)

Flashing Process Interrupted

  • Do not disconnect the device during flashing
  • Ensure stable USB connection
  • Close other applications that might be using the serial port
  • Retry the flashing process

Device Not Responding After Flash

  • Perform a factory reset if available
  • Verify the correct firmware file was flashed for your device model
  • Check device power supply and battery status
  • Contact support if issues persist
